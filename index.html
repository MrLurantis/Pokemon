什么是 Pokemon

Pokemon (任天堂掌机游戏系列)

《精灵宝可梦》，另有常见非官方译名：《口袋妖怪》(中国大陆译名)、《宠物小精灵》(中国香港译名)、 《神奇宝贝》（中国台湾译名），是由Game Freak和Creatures株式会社开发，任天堂发行的系列游戏，是 精灵宝可梦媒体作品的主要部分。最初的作品是在1996年于日本发行的Game Boy角色扮演游戏。精灵宝可梦 系列一直贯穿任天堂的各代掌机之中。系列中的游戏通常成对发售（有所区别），复刻版常出现在初版游戏 发布的几年后。游戏的主系列由角色扮演类游戏组成，旁支系列的游戏则包含有其他的类型，例如动作角色 扮演游戏、益智游戏和电子宠物等。

精灵宝可梦系列是世界上第二热销的系列电子游戏，仅次于任天堂的超级马里奥系列。

2016年2月26日，在精灵宝可梦直面会上，石原恒和公开了主系列的第七世代新作《精灵宝可梦太阳·月亮》， 于2016年11月18日发售。 本作品内置9种语言选择，其中首次包括了简体中文和繁体中文。另外石原恒和还 宣布了精灵宝可梦这一官方译名。

中文名 精灵宝可梦 开发商 GAME FREAK
原版名称 ポケットモンスター 发行商 任天堂，精灵宝可梦股份有限公司
中文名 精灵宝可梦 开发商 GAME FREAK
其他名称 Pokémon 制作人 田尻智，杉森建，増田順一（音乐），石原恒和
游戏类型 角色扮演、奇幻冒险 原 作 田尻智
游戏平台 Game Boy 编 剧 石原恒和
Game Boy Color 音 乐 增田顺一，宫崎慎二，景山将太
Game Boy Advance 内容主题 收集，对战，交换
任天堂DS 玩家人数 多人
任天堂3DS 最新版本 精灵宝可梦究极太阳/究极月亮
地区 日本

历代版本

第一世代

第一世代（gen1）作品：红、绿、蓝、黄
最初的精灵宝可梦游戏是带有策略元素的日本角色扮演游戏，由田尻智为Game Boy扮演游戏和其续集、 重制版被认为是精灵宝可梦的主系列游戏，为大多数系列的粉丝所熟知。

第二世代

第二时代（gen2）作品：金、银、水晶
精灵宝可梦系列游戏的第二时代开始自1999年Game Boy Color游戏《精灵宝可梦金·银》在日本的发售。 游戏在澳洲及北美于2000年10月发售，欧洲版于2001年4月发售。与先前的世代相似，增强重制版 《精灵宝可梦水晶》在之后发售。

第三世代

第三世代（gen3）作品：红宝石、蓝宝石、绿宝石、火红、叶绿
在Game Boy Advance游戏《精灵宝可梦红宝石·蓝宝石》于2002年发售时，精灵宝可梦系列进入了 第三世代，其后还发售了Game Boy Advance的《精灵宝可梦红·绿》复刻版《精灵宝可梦火红·叶绿》 （红绿指日本最初的第一世代游戏；日本以外的地区则是红蓝）。《精灵宝可梦红宝石·蓝宝石》的 增强重制版《精灵宝可梦绿宝石》随后也进行了发售。

第四世代

第四世代（gen4）作品：钻石、珍珠、白金、心金、魂银
2006年，随着任天堂DS游戏《精灵宝可梦钻石·珍珠》在日本的发售，系列进入了第四世代。 其余的第四世代主系列游戏则包括钻石珍珠的加长版，与《黄》《水晶》《绿宝石》地位相同的 《精灵宝可梦白金》。[11-12] 该作为任天堂DS游戏，在日本于2008年9月13日发售。 第二世代游戏《精灵宝可梦 金·银》同样发布了任天堂DS复刻版《精灵宝可梦心金·魂银》。 最初在日本于2009年9月12日发售，后在2010年3月间于北美、澳洲和欧洲发售。

第五世代

第五世代（gen5）作品：黑、白、黑2、白2
这一世代引入了156种新精灵宝可梦（自比克提尼起至盖诺赛克特止）。 第四世代旁支游戏《乱战！精灵宝可梦大纷争》的续作，旁支游戏《超级精灵宝可梦大纷争》 在2011年8月11日于日本发售。

第六世代

第六世代（gen6）作品：X、Y、终级红宝石、始源蓝宝石
2013年1月8日，岩田聪公布了将于2013年10月12日全球发售的任天堂3DS游戏《精灵宝可梦X·Y》， 宣布了第六世代的开始。XY版进行了完全的3D渲染，然而只有游戏中的特定部分能够进行立体3D展示。
2014年5月7日，任天堂在宣传片中公布了第三世代作品《精灵宝可梦红宝石·蓝宝石》的复刻版， 《精灵宝可梦 终极红宝石·始源蓝宝石》。两版游戏于2014年11月21日全球发售。

第七世代

第七世代（gen7）作品：太阳、月亮、究极之日、究极之月
在2016年2月26日的精灵宝可梦直面会上，石原恒和公开了系列的第七世代续作《精灵宝可梦太阳·月亮》 ，于2016年11月18日全球同步发售。[2] 作品内置9种语言供全球玩家选择，其中首次包括了中文， 并有繁简两种可供选择。
